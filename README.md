**BlogCMS - Backend & Dashboard Admin**
Système de gestion de blog complet avec tableau de bord administrateur développé en PHP procédural.
1. Fonctionnalités
- **Pour les Administrateurs**
Dashboard avec statistiques
Gestion complète des catégories (CRUD)
Modération des commentaires
Gestion des utilisateurs et rôles

- **Pour les Auteurs**
Créer, éditer et supprimer leurs articles
Visualiser leurs publications
Poster des commentaires

- **Pour les Visiteurs**
Consulter les articles publiés
Commenter les articles

2. Technologies Utilisées

Backend : PHP 8 (procédural)
Base de données : MySQL avec PDO
Frontend : HTML5, CSS3, Tailwind CSS
